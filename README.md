# Consumo de API de Filmes com C#

Este repositório contém um exercício prático desenvolvido durante o curso de C#, focado no consumo de dados externos e manipulação de arquivos JSON.

## 🚀 O que este projeto faz:
- Utiliza `HttpClient` para realizar requisições assíncronas.
- Consome um endpoint que retorna uma lista de filmes em formato JSON.
- Realiza a desserialização dos dados para objetos C# usando `System.Text.Json`.
- Exibe as informações formatadas no console.

## 🛠️ Tecnologias Utilizadas:
- .NET 10.0
- C#
- System.Text.Json

## 📚 Aprendizados:
- Manipulação de classes com atributos de serialização (`[JsonPropertyName]`).
- Tratamento de erros em requisições HTTP com blocos `try-catch`.
- Uso de métodos assíncronos (`async/await`).
